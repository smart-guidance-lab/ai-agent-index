# AI Agent Index

A curated directory of AI agents and assistants for developers and technical users.

---

## Development & Code

### Cursor Rules
Create and manage Cursor rules for persistent AI guidance. Use when you want coding standards, project conventions, or RULE.md files.

### Agent Skills
Guides for creating effective Agent Skills. Use when authoring a new skill or learning SKILL.md format and best practices.

### Settings
Modify Cursor/VSCode user settings in settings.json. Use for editor preferences, themes, keybindings, or format on save.
